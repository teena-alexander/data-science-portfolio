# Olympics Data Analysis

## Goal
Analyze Olympics data to identify trends in medal counts, country performance, and athlete statistics.

## Tools Used
- Python (Pandas, NumPy, Matplotlib, Seaborn)
- Jupyter Notebook

## Approach
- Cleaned and explored the dataset
- Calculated total medals by country and year
- Visualized trends using bar charts and line plots
- Performed basic statistical analysis to identify patterns

## Key Insights
- USA consistently performs well in overall events
- Medal trends over the years show an increase in participation from new countries
- Correlation found between athlete age and performance in certain events
